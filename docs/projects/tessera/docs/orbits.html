Euro pack extension

Euro pack — additional characters for European writing systems (à á ä å č ć ď đ è é ě ì í ł ń ò ó ô ö ř ś ť ù ú ů ü ý ź ż). Where a Euro character is phonetically equivalent to a base character, it is shown as a variant reading of the base glyph (e.g. ɛ/ä/ě, ō/å) rather than a new orbit. The 23 genuinely new Euro characters occupy the four previously-reserved diagonal orbits and all 19 remaining octuplets, exhausting all free slots.
The +centre rule. Adding the central pixel maps every short vowel to its long form (e → ē, i → ī, u → ū, a → ā, o → ō) and every dental consonant to its retroflex counterpart (t → ṭ, d → ḍ, n → ṇ, s → ṣ, l → ḷ, z → ẓ, r → ṛ). This rule applies only within the base alphabet; Euro pack characters carry no such systematic relationship to one another.
