The Tessera, all 110 cells

Each cell hosts one orbit (or one chiral sub-orbit, for the brackets). Top-left: the character(s) it represents — multiple when several scripts share the orbit. Top-right: the canonical 3 × 3 pixel grid. Centre: the calligraphic form, computed live by the stroke-derivation rules. Bottom-left: the canonical g-value (the position in the 512-long ordered list of grids). Bottom-right: the orbit name. 104 cells are assigned; the last 6 are left for whatever comes next.
