topo-inside — mobile preview

The touch build — on-screen thumb controls and mobile level-of-detail — running in a phone-sized frame. This is the same page a phone loads; it forces the mobile path with the #mobile URL hash.

·
Drag inside the screen to look; use the on-screen buttons to fly, rise / descend, change speed and hover. On a real phone, open index.html directly. Serve this over the dev server (python3 serve.py) — opening it as a file:// will not load the module.
